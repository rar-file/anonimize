### Added
- Initial release with core anonymization functionality
- Support for multiple anonymization strategies: replace, hash, mask, remove
- CSV and JSON file anonymization
- Database anonymization (SQLite, PostgreSQL, MySQL)
- PII detection with regex and heuristic methods
- Relationship preservation for consistent anonymization
- Batch processing for large datasets
- Comprehensive test suite with 90%+ coverage
- GitHub Actions CI/CD pipelines
- Documentation with MkDocs
- 24+ usage examples
